Projects | Publications | Resume

Dementia AR

An AR empathy-training tool that simulates visual symptoms of dementia to help caregivers better understand and support their patients. Built with Unity and Meta Quest, using LUT-based passthrough distortion to replicate optic nerve degradation.

Lunar Digital Twin

A high-fidelity VR twin of the Apollo 17 landing site, built for ASU’s Digital Discovery Initiative. Enables immersive exploration of real lunar terrain using photogrammetry, GIS data, and Cesium integration to support planetary science and remote collaboration.

Sensory Augmentation for Robot Operators

An AR application developed during my PTC internship, enhancing Human-Robot Interaction by visualizing robotic sensor data in Augmented Reality. This project emphasizes usability in advanced control systems through AR integration.

AR Magnetic Field Visualizer

A custom-built device that visualizes magnetic fields in Augmented Reality. Using AR, microcontrollers, and sensors, it transforms abstract physics into an engaging educational tool, giving students the superpower to see the invisible.

PneuTouch

A pneumatic VR haptic interface allowing users to grasp virtual objects of varying shapes. Published in the CHI workshop on sustainable haptics, this project blends research with practical VR solutions.

Zen Gardeners

A multiplayer, co-localized AR game where users create Japanese Zen Gardens. Utilizing Azure Remote Rendering, it enables real-time collaboration with high-quality models rendered seamlessly.

VR Electromagnetism Lab

An educational VR application that shrinks students to the atomic level to manipulate electrons and protons, offering a unique view of electromagnetic fields. This project creates immersive, educational VR experiences.

Latent Space Exploration in MR

An XR+AI app for exploring 6-dimensional latent spaces of a Variational Autoencoder. By integrating machine learning with spatial computing, it demonstrates XR design and development's potential.

AR Window

An Augmented Reality app turning the Hololens 2 into a MIDI controller for unique music creation. This project merges music production with AR for innovative musical tools.

Hololens MIDI Controller

An Augmented Reality app turning the Hololens 2 into a MIDI controller for unique music creation. This project merges music production with AR for innovative musical tools.

Pin That GIF!

A webXR app allowing users to place persistent GIFs at VPS wayspots, creating a shared social canvas. Runner-up in Niantic’s VPS for Web challenge, it combines WebXR with API integration.

AR Paint

A relaxing AR drawing app that won the “Icarus” award at Sunhacks 2021 Hackathon. This project showcases fun, engaging AR applications.

AR Covid Visualizer

A data visualization tool offering a new perspective on the early COVID pandemic through Augmented Reality. This project uses AR for impactful data storytelling.

3D C# Game Engine

A custom-built 3D game engine in C# created for a game engine development course. The video showcases an Air hockey game with custom enemy AI, highlighting my skills in game engine development and AI programming.
