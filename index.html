Sensory Augmentation for Robot Operators

An innovative AR application developed during my internship with PTC, enhancing Human-Robot Interaction by visualizing robotic sensor data in Augmented Reality. Integrating AR with robotics, this project emphasizes the usability of cutting-edge control systems, showcasing my focus on user experience.

AR Magnetic Field Visualizer

A custom-built device to visualize magnetic fields in Augmented Reality, giving students the superpower to see the invisible. Using AR, microcontrollers, and sensor integration, we transform abstract physics into an engaging, educational experience. This flagship project encapsulates my vision as a creative developer, merging technology and imagination to unveil unseen realities.

PneuTouch

A pioneering pneumatic VR haptic interface that lets users grasp virtual objects of varying shapes and sizes, delivering a unique tactile experience. Published in the CHI workshop on sustainable haptics, this project showcases my ability to blend research with practical VR solutions.

Zen Gardeners

A multiplayer, co-localized AR game where users collaborate to create beautiful Japanese Zen Gardens. Utilizing Azure Remote Rendering, this project enables real-time collaboration with high-quality models rendered seamlessly, highlighting my expertise in cloud computing and AR development.

VR Electromagnetism Lab

An educational VR application that shrinks students to the atomic level to manipulate and combine electrons and protons, offering a unique view of electromagnetic fields. This project demonstrates my ability to create immersive, educational VR experiences.

Latent Space Exploration in MR

An XR+AI app that allows users to explore 6-dimensional latent spaces of a custom-trained Variational Autoencoder. By integrating machine learning with spatial computing, this project reveals the vast potential of XR design and development, enabling creative, generative outputs.

AR Window

A Hololens 2 application for placing, customizing, and measuring realistic windows in homes. Created for a psychology research study, this project emphasizes my skills in client communication and requirements gathering, combined with advanced AR technology.

Hololens MIDI Controller

An Augmented Reality app turning the Hololens 2 into a MIDI controller for unique music creation. This project blends my passion for music production with AR, providing an innovative tool for musicians.

Pin That GIF!

A webXR app that allows users to place persistent GIFs at VPS wayspots, creating a shared social canvas. Runner-up in Niantic’s VPS for Web challenge, this project demonstrates my skill in combining WebXR with API integration and creative user engagement.

AR Paint

A relaxing AR drawing app that won the “Icarus” award at the Sunhacks 2021 Hackathon. This project showcases my ability to create fun, engaging AR applications.

AR Covid Visualizer

A data visualization tool that offers a new perspective on the early months of the COVID pandemic through Augmented Reality. This project highlights my ability to use AR for impactful data storytelling.
